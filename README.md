# PlasmaDonate - Оптимізація продуктивності 

## Завдання 1. Аналіз застосунку

| Аспект | Вибір | Чому |
|--------|-------|------|
| Анімація | Вкладка "Новини" | Картки новин з'являються з анімацією |
| Оптимізація рендерів | Компонент Card | Перерендерюється при кожній зміні теми |
| Великі залежності | moment, lodash | Відсутні, замінено на dayjs |

## Завдання 2. Анімація (Reanimated)

- **Файл:** `screens/PlasmaNewsScreen.tsx`
- **Анімація:** `ZoomIn` з затримкою 100ms між картками
- **Відео:** `svideo/animation-dark.mov` та `video/animation-light.mov`

## Завдання 3. Оптимізація рендерів

| Файл | Оптимізація |
|------|-------------|
| `components/Card.tsx` | `React.memo` |
| `screens/HomeScreen.tsx` | `useCallback` для 5 функцій |

## Завдання 4. Зменшення ваги застосунку

| Перевірка | Результат |
|-----------|-----------|
| `moment` | ❌ відсутній (не використовується) |
| `lodash` | ❌ відсутній (не використовується) |
| `dayjs` | ✅ встановлено (легка альтернатива) |
